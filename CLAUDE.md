# Portfolio Website — Claude Code Guide

## Project Overview
Personal portfolio website for Wong Huai Wen (Raymond).
Terminal/CRT aesthetic built with pure HTML, CSS, and vanilla JavaScript.

## Live URL
https://raymond-520.github.io/portfolio-website/

## File Structure
- index.html — full single-page portfolio
- styles.css — all styles and design tokens
- app.js — animations, scroll effects, terminal boot sequence
- src-backup/ — original React version (kept for reference)

## Design System
- Background: #161513
- Accent: #f97316 (burnt orange)
- Text: #f5f0e8 (warm beige)
- Fonts: Space Grotesk (display) + JetBrains Mono (mono)

## Dev Server
npm run dev

## Deploy to GitHub Pages
npm run deploy

## Commit Convention
- feat: new features
- fix: bug fixes
- style: UI changes
- docs: documentation
- refactor: code changes

## Key Sections to Update
- Projects: index.html → .projects section
- Experience: index.html → #experience section
- Skills: index.html → .skill-cols section
